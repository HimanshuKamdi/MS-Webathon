# Setting Up Backend

## Run these commands in main directory
### `pip install virtualenv`
### `.\env\Scripts\activate`
### `pip install django djangorestframework django-cors-headers`
### `cd .\backend\`
### `python manage.py runserver`


# Setting Up Frontend
## Run these commands in main directory
### `cd .\frontend\`
### `npm i`
### 'npm start'
