# Setting Up Backend
## Run these commands in main directory
### `pip install virtualenv`
### `virtualenv env`
### `.\env\Scripts\activate`
### `pip install -r requirements.txt`
### `cd .\backend\`
### `python manage.py runserver`


# Setting Up Frontend
## Run these commands in main directory
### `cd .\frontend\`
### `npm config set legacy-peer-deps true`
### `npm i`
### `npm start`
